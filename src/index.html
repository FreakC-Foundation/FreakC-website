FreakC is a Batch-transpiled multi-paradigm scripting language. The language is flexible and easy to learn. It solves the problems that you're having with Batch, and it's also a good language on its own.

FreakC is 100% open-source, so if you want to create your own language, you totally can by having a look on Github
