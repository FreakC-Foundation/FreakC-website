FreakC is a programming language that uses dirty syntax that you may only use while you are drunk. It is transpiled to Batch, and... if you want to create something with Batch but want something idiotic and straight-forward, try FreakC.

FreakC is 100% open-source, so if you want to create your own dirty language, you totally can by having a look on Github
